# Legend Of Indle — V150 Ultimate Local MMORPG Lab

Esta build consolida o pedido de evoluir o jogo até o V150, usando o material enviado como inspiração técnica/visual.

## Implementado no cliente

- Botão `🔥 V150` no menu.
- Central V150 com abas:
  - Batalha
  - Visual
  - Mundo
  - Sistemas
  - Roadmap V96–V150
- Battle Director visual mais agressivo.
- Inimigo com fallback visual e barra no palco.
- Target ring e linha de combate.
- Companheiros/pets visuais no campo.
- Minions/sombras orbitando o inimigo.
- Skill automática visual a cada poucos segundos.
- Controles V150 para testar boss, golpe pesado, VFX e loot.

## Correção importante

Mesmo se o sprite do monstro falhar, o jogo agora desenha um inimigo procedural visível no Canvas.

## Próximas frentes reais

- Trocar o fallback por sprites próprios finais.
- Criar assets PNG de cabelo/roupa/arma/asas.
- Evoluir mapa e zonas para um modo campanha jogável.
- Empacotar em launcher/executável local.
