# Reference Imports / Adaptação V93

Esta iteração foi orientada para "pegar o que há de melhor do visual e animações do material enviado e adaptar para dentro do Legend Of Indle".

## O que foi absorvido na adaptação
- Sensação de MMORPG dark fantasy no fluxo de entrada e interface.
- Organização visual inspirada em clientes antigos com:
  - bolsa/slots,
  - janela de personagem,
  - foco em combat feel,
  - preparação para mais camadas visuais.
- Estrutura pronta para evolução de:
  - fashion por camadas,
  - animações com pseudo-rig,
  - atalhos,
  - forja,
  - skillbook.

## Resultado prático nesta build
- nova bolsa multiaba;
- painéis e cards mais ricos;
- pseudo-animação segmentada para herói/monstro;
- mais ênfase em skills, equipamento e progressão.
